Attributes vs Props
* Attributes are HTML element properties like class,id,href used to define element behavior or style.
* Props are inputs passed to React components to customize their rendering and behavior.
* Attributes are static and controlled by the browser, while props are dynamic and controlled by React components.

Children Prop
* Children prop allows passing nested elements inside a React component.
* It is accessed via props.children in functional or class components.
* Children can be single or multiple elements, and React provides helpers to work with them.
